# PowerShell Toolkit RA

Repository che raccoglie moduli PowerShell e script di supporto per:
- connessioni cloud (Microsoft Graph, Microsoft Teams, ecc.)
- utilità su file ed export
- strumenti DNS
- helper per UX e automazioni

Il progetto è organizzato in moduli indipendenti ed è pensato per essere
scalabile, riutilizzabile e manutenibile nel tempo.
